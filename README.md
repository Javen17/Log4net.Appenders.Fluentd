# Log4net.Appenders.Fluentd

----------
Fluentd appender for Log4net.
 
## Installation

    PM> Install-Package Log4net.Appenders.Fluentd

### Dependencies

    - log4net
    - MsgPack.Cli

### Building the project

    dotnet build
